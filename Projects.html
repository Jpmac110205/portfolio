Black Jack

This is a simple black jack game that I made using Java, and Swift. It is a single-player game where you can play against the dealer. The game is designed to be easy to use and understand, with a clean layout that makes it easy to play and understand. This game has a working dealer which abides by the rules of the game, a fully functioning betting system and the physical cards that the player recieves is displayed. You can find the source code on my GitHub page.
